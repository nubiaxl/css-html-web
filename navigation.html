Browser Compatibility

Good
  • Displays on current versions of Internet Explorer (6+)
  • Displays on current versions of Firefox (2+)
  • Displays on current versions of Netscape (7+)
  • Displays on current versions of Opera (9+)
Bad
  • Displays on current versions of Safari (both Mac and Windows)

Navigation

Good
  • Main navigation links are clearly and consistently labeled
  • Navigation is easy to use for target audience
  • If image, Flash, or DHTML is the main navigation, clear text links are in the footer section of the page (accessibility)
  • Navigational aids, such as site map, skip navigation link, or breadcrumbs, are used
Bad
  • None
